## About

React Hooks provide a powerful and expressive way to manage state, side effects, and reusable logic in functional React components. This repository demonstrates practical usage of hooks in multiple projects, including **state management, context, side effects, custom hooks, and performance optimization**.

---

## Projects

Each folder in this repository contains a separate project or example:

### **1. useState Examples**
- Demonstrates how to use `useState` for managing local component state.
- Includes examples with forms, counters, and toggles.

### **2. useEffect Examples**
- Shows how to use `useEffect` for side effects like API calls, subscriptions, and timers.
- Explains dependencies array and cleanup functions.

### **3. useContext Examples**
- Illustrates global state management using `useContext`.
- Includes context providers and consumers for reusable state.

### **4. Custom Hooks**
- Demonstrates how to create reusable hooks for logic like form handling, fetching data, and timers.
- Encourages modular and maintainable code.

### **5. Advanced Hooks**
- Examples of `useReducer`, `useRef`, `useMemo`, `useCallback`.
- Focused on performance optimization and complex state management.

### **6. Complete Projects**
- Full small applications using multiple hooks together.
- Shows real-world usage of hooks in functional components.

> Each project folder contains a `README.md` explaining its own code, usage, and learning points.
